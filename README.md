# 🚦 Density-Based Traffic Light Control System

## 📌 Description
This project implements a smart traffic light control system that dynamically adjusts signal timings based on real-time vehicle density. Unlike traditional fixed-timer systems, this approach reduces congestion and improves traffic flow efficiency.

---

## 🚀 Features
- Real-time vehicle detection using computer vision  
- Dynamic traffic signal control based on density  
- Reduces waiting time and congestion  
- More efficient than fixed-time traffic systems  

---

## 🛠️ Technologies Used
- Python  
- OpenCV  
- NumPy  

---

## ⚙️ How It Works
1. Captures video input (live camera or recorded video)  
2. Uses OpenCV to detect vehicles in each frame  
3. Counts the number of vehicles on the road  
4. Adjusts traffic signal timing based on vehicle density  

---

### 🔹 Step 1: Install Python
