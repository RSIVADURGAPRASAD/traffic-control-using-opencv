# 🚦 Density-Based Traffic Light Control System

## 📌 Description
This project implements a smart traffic light system that adjusts signal timings based on real-time vehicle density using computer vision techniques.

## 🚀 Features
- Vehicle detection using OpenCV  
- Dynamic signal timing  
- Reduces traffic congestion  

## 🛠️ Technologies Used
- Python  
- OpenCV  
- NumPy  

## ▶️ How to Run
pip install opencv-python numpy  
python main.py
